# 건강 블로그 글쓰기 가이드

> 모든 포스트 작성 시 반드시 이 가이드를 따르세요.

## 1. 글 구조 가이드

### 제목
- 구체적 숫자 포함: "7가지", "월 100만원", "2026년"
- 예: "허리 통증 줄이는 스트레칭 5가지 — 하루 10분이면 충분해요"

### 도입부
- 일상과 연결: "~해본 적 있으시죠?", "~고민이셨죠?"
- 1~2문단, 독자의 상황에 공감

### 본문 소제목 구성
- 정보성 소제목 2개 + 개인 의견/액션 소제목 1개
- 각 섹션: 핵심 포인트 → 구체적 예시/데이터 → 실행 가능한 팁

### 마무리
- 독자가 "지금 당장" 할 수 있는 액션 아이템 제시

### 포맷
- 한 줄 80~100자 이내
- 문장이 25자 넘으면 끊기

---

## 2. 한국어 문체 가이드

### 금지 패턴 → 대체 패턴

| 금지 | 대체 |
|------|------|
| ~합니다, ~할 수 있습니다 | ~거든요, ~더라고요, ~예요/이에요 |
| 획기적으로 | 삭제 또는 구체적 수치로 |
| 극대화하고 | 늘리고, 높이고 |
| 현명한 전략입니다 | 이렇게 하면 돼요 |
| 가장 중요합니다 | 핵심은 이거예요 |
| 큰 도움이 됩니다 | 바로 효과가 나요 |
| 주의가 필요합니다 | 이건 조심하세요 |
| ~하는 것이 좋습니다 | ~하세요 |
| ~라고 할 수 있겠습니다 | 삭제 |

### 불필요한 수식어 제거
- "매우", "정말로", "굉장히", "상당히" → 삭제

### 문장 길이
- 25자 넘으면 두 문장으로 나누기

## 3. 건강 블로그 특화 규칙

- **author:** 오늘도 건강
- 의료 정보는 권위 유지하되 도입부/마무리는 대화체
